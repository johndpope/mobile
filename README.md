## Build Setup

``` bash
# instalar as dependencias
npm install

# roda um server (dev) em localhost:8080
npm run dev

# faz a build com minify
npm run build

# faz a build com bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
